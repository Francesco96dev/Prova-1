Css di base

Il CSS è un linguaggio di stile (non programmazione) che definisce l'aspetto visuale delle pagine web mentre il contenuto è strutturato da HTML

Il CSS consente di separare il contenuto dalla presentazione, migliorando la manutenibilità e la leggibilità del codice. "Cascading Style Sheets" significa "fogli di stile a cascata" Cascading indica che gli stili possono essere applicati in modo gerarchico, con priorità basata su specificità e ordine di applicazione.

dove scrivere il CSS

Il CSS può essere scritto in tre modi principali: inline, interno ed esterno.

  • Inline: Gli stili sono applicati direttamente agli elementi HTML tramite l'attributo "style".
  • Interno: Gli stili sono definiti all'interno di un tag style nella sezione head del documento HTML.
  • Esterno: Gli stili sono scritti in un file CSS separato e collegati al documento HTML tramite il tag ``.

Immagine di background

L'immagine di background può essere impostata utilizzando la proprietà CSS "background-image". Può essere applicata a qualsiasi elemento HTML, come ad esempio il body o un div.
